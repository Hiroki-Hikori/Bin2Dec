# Bin2Dec

This is a small project to convert a binary number to a decimal number with C++. To run the code, you can use this following commands.

Clone the Project:

```
git clone https://github.com/Hiroki-Hikori/Bin2Dec/
```

```bash
cd Bin2Dec
chmod +x Bin2Dec
./Bin2Dec
```

You can also view the original code by opening the `main.cpp` file.
